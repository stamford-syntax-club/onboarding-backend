## Install dependencies

```bash
npm install
```

### Run project

```bash
npm run dev
```

### Start MongoDB and S3 Services

```bash
docker compose up -d
```

### Stop the MongoDB and S3 Services

```bash
docker compose down
```

Access the site via [http://localhost:8080](http://localhost:8080)
